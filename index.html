Repo Shape

What belongs here

  • New artifacts should live under artifacts/<slug>/ with local files scoped to that asset.
  • Shared repo files stay at the root. Artifact logic stays with the artifact.
  • Markdown-only artifacts are valid when the source format itself is what needs to be published.
  • The root index should stay lightweight and point outward, not absorb feature work.
